<summary>문제 38. 다음 선형검색 코드의 출력 결과는?</summary>

출처: `IMG_8871`, 38번

```python
def seq_search(a, key):
    for i in range(len(a)):
        if a[i] == key:
            return i
    return -1

print(seq_search([6, 4, 8, 1, 3], 1))
```

1. 1
2. 2
3. 3
4. 4

<details>
<summary>해설 보기</summary>

- 정답: 4번
- 이유: 값 `1`은 인덱스 `3`에 있으므로 반환값은 `3`이다. 보기의 번호는 네 번째가 정답이다.

</details>
</details>

<summary>문제 40. 다음 보초법 코드에서 반복문 내부에 검사하는 조건은 몇 개인가?</summary>

출처: `IMG_8871`, 40번

```python
def seq_search_sentinel(a, key):
    a.append(key)
    i = 0
    while True:
        if a[i] == key:
            break
        i += 1
    a.pop()
```

2. 1개
3. 2개
4. 3개

<details>
<summary>해설 보기</summary>

- 정답: 2번
- 이유: 보초 값을 끝에 붙였기 때문에 반복문 내부에서는 `a[i] == key` 한 조건만 보면 된다.

</details>

<summary>문제 41. 다음 이진검색 코드의 출력 결과는?</summary>

출처: `IMG_8871`, 41번

```python
def bin_search(a, key):
    pl = 0
    pr = len(a) - 1
    while pl <= pr:
        pc = (pl + pr) // 2
        if a[pc] == key:
            return pc
        elif a[pc] < key:
            pl = pc + 1
        else:
            pr = pc - 1
    return -1

a = [3, 8, 15, 21, 27, 32, 40]
print(bin_search(a, 27))
```

1. 3
2. 4
3. 5
4. -1

<details>
<summary>해설 보기</summary>

- 정답: 2번
- 이유: 정렬 배열에서 `27`은 인덱스 `4`이므로 반환값은 `4`다.

</details>
</details>
