# 문제은행 허브

이제 문제은행은 개념별로 관리한다. 각 파일 안에 문제 원문과 해설을 함께 넣고, 개념노트에서는 접어서 펼쳐 볼 수 있게 연결한다.

## 읽는 순서

1. 개념노트를 읽다가 해당 문제은행을 바로 펼쳐 본다.
2. 문제를 먼저 풀고, 같은 토글 안의 해설을 확인한다.
3. 필요하면 해당 개념의 `원본 문제` 문서를 열고, 마지막 확인용으로 사진 원본을 본다.
