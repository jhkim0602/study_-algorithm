### 문제 42
다음 코드의 출력 결과는?

```python
x = '0123456789ABCDEFGHIJKLMNOPQRSTUVWXYZ'

def card_conv(n, r):
    d = ''
    while n > 0:
        d += x[n % r]
        n //= r
    return d[::-1]

print(card_conv(10, 2))
```

① 10
② 1010
③ 1001
④ 1111

### 문제 43
다음 코드의 출력 결과는?

```python
count = 0
for n in range(2, 8):
    for i in range(2, n):
        count += 1
        if n % i == 0:
            break
    else:
        print(n, end=' ')
```

① `2 3 5 7`
② `2 3 4 5 6 7`
③ `3 5 7`
④ `2 5 7`

### 문제 45
다음 코드의 시간복잡도로 가장 적절한 것은?

```python
for n in range(2, N):
    for i in range(2, n):
        if n % i == 0:
            break
```

① `O(log n)`
② `O(n)`
③ `O(n log n)`
④ `O(n^2)`
