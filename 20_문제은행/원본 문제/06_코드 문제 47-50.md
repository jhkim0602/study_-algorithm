### 문제 47
다음 코드의 출력 결과는?

```python
def hash_func(key):
    return key % 5

table = [[] for _ in range(5)]
keys = [7, 12, 17]

for k in keys:
    h = hash_func(k)
    table[h].append(k)

print(table)
```

① `[[], [], [7, 12, 17], [], []]`
② `[[], [], [], [7, 12, 17], []]`
③ `[[], [7], [12], [17], []]`
④ `[[], [], [7], [12], [17]]`

### 문제 49
다음 코드의 출력 결과는?

```python
def hash_func(key):
    return key % 5

table = [None] * 5

def insert(key):
    h = hash_func(key)
    while table[h] is not None:
        h = (h + 1) % 5
    table[h] = key

keys = [7, 12, 17]
for k in keys:
    insert(k)

print(table)
```

① `[None, None, 7, 12, 17]`
② `[None, None, 17, 7, 12]`
③ `[None, None, 7, 17, 12]`
④ `[7, 12, 17, None, None]`
