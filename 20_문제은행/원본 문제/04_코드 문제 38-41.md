### 문제 38
다음 선형검색 코드의 출력 결과는?

```python
def seq_search(a, key):
    for i in range(len(a)):
        if a[i] == key:
            return i
    return -1

print(seq_search([6, 4, 8, 1, 3], 1))
```

① 1
② 2
③ 3
④ 4

### 문제 40
다음 보초법 코드에서 반복문 내부에서 검사하는 조건은 몇 개인가?

```python
def seq_search_sentinel(a, key):
    a.append(key)
    i = 0
    while True:
        if a[i] == key:
            break
        i += 1
    a.pop()
```

① 0개
② 1개
③ 2개
④ 3개

### 문제 41
다음 이진검색 코드의 출력 결과는?

```python
def bin_search(a, key):
    pl = 0
    pr = len(a) - 1
    while pl <= pr:
        pc = (pl + pr) // 2
        if a[pc] == key:
            return pc
        elif a[pc] < key:
            pl = pc + 1
        else:
            pr = pc - 1
    return -1

a = [3, 8, 15, 21, 27, 32, 40]
print(bin_search(a, 27))
```

① 3
② 4
③ 5
④ -1
