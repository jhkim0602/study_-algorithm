### 문제 33
다음 코드의 출력 결과는?

```python
def max_of(a):
    maximum = a[0]
    for i in range(1, len(a)):
        if a[i] > maximum:
            maximum = a[i]
    return maximum

print(max_of([4, 1, 9, 3, 7]))
```

① 4
② 7
③ 9
④ 24

### 문제 34
다음 `max_of()` 함수에서 배열 길이가 `n`일 때, 비교 연산 `a[i] > maximum`은 총 몇 번 수행되는가?

```python
def max_of(a):
    maximum = a[0]
    for i in range(1, len(a)):
        if a[i] > maximum:
            maximum = a[i]
```

① `n`
② `n - 1`
③ `n / 2`
④ `log n`

### 문제 35
다음 코드의 출력 결과는?

```python
def reverse(a):
    n = len(a)
    for i in range(n // 2):
        a[i], a[n - i - 1] = a[n - i - 1], a[i]

x = [1, 2, 3, 4, 5]
reverse(x)
print(x)
```

① `[5, 4, 3, 2, 1]`
② `[2, 1, 4, 3, 5]`
③ `[1, 2, 3, 4, 5]`
④ `[5, 4, 3, 1, 2]`

### 문제 36
다음 역순 알고리즘의 시간복잡도는?

```python
def reverse(a):
    n = len(a)
    for i in range(n // 2):
        a[i], a[n - i - 1] = a[n - i - 1], a[i]
```

① `O(1)`
② `O(log n)`
③ `O(n)`
④ `O(n^2)`
