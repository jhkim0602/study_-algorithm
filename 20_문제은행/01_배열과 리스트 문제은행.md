<details>
<summary>문제 21. 자료구조(data structure)에 대한 설명으로 가장 적절한 것은?</summary>

출처: `IMG_8869`, 21번

1. 데이터를 화면에 출력하는 방법
2. 데이터를 저장하는 메모리 크기
3. 데이터를 효율적으로 관리하기 위한 데이터의 조직 방식
4. 데이터를 입력받는 방법

<details>
<summary>해설 보기</summary>

- 정답: 3번
- 이유: 자료구조는 데이터를 어떤 모양으로 조직해 접근, 검색, 삽입, 삭제를 더 효율적으로 만들지에 대한 설계다. 출력 방식이나 입력 방식은 자료구조의 정의가 아니다.
- 연결 포인트: 배열과 리스트 단원은 단순 문법이 아니라, 데이터를 한 덩어리 구조로 다루는 첫 출발점이다.

</details>

출처: `IMG_8868`, 14번

<details>
<summary>해설 보기</summary>

- 연결 포인트: 음수 step과 슬라이싱 경계 감각은 이후 배열 응용 문제의 바탕이 된다.

</details>

<details>
<summary>문제 37. 다음 슬라이싱 결과는?</summary>

출처: `IMG_8870`, 37번

```python
a = [10, 20, 30, 40, 50, 60]
print(a[1:5:2])
```

1. `[10, 30, 50]`
2. `[20, 40]`
3. `[20, 30, 40, 50]`
4. `[60, 40, 20]`

<details>
<summary>해설 보기</summary>

- 정답: 2번
- 이유: 시작 인덱스는 `1`, 끝 인덱스 `5`는 포함하지 않고, `2`칸씩 건너뛰므로 인덱스 `1`, `3`의 값인 `20`, `40`이 나온다.
- 연결 포인트: 슬라이싱은 시작, 끝, 간격을 따로 읽는 습관이 중요하다.

</details>
</details>

<summary>문제 22. 다음 기수 변환 코드에서 마지막에 <code>[::-1]</code>을 사용하는 이유는?</summary>

출처: `IMG_8869`, 22번

```python
x = '0123456789ABCDEFGHIJKLMNOPQRSTUVWXYZ'

def card_conv(n, r):
    d = ''
    while n > 0:
        d += x[n % r]
        n //= r
    return d[::-1]
```

1. 정렬하기 위해
2. 중복을 제거하기 위해
3. 나머지를 구한 순서가 거꾸로이기 때문에
4. 문자열을 리스트로 바꾸기 위해

<details>
<summary>해설 보기</summary>

- 정답: 3번
- 이유: 진수 변환에서는 낮은 자리수의 나머지가 먼저 나오므로, 최종 표현을 위해 역순 복원이 필요하다.
