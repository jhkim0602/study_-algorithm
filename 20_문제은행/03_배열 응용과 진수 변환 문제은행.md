<summary>문제 22. 다음 기수 변환 코드에서 마지막에 <code>[::-1]</code>을 사용하는 이유는?</summary>

출처: `IMG_8869`, 22번

```python
x = '0123456789ABCDEFGHIJKLMNOPQRSTUVWXYZ'

def card_conv(n, r):
    d = ''
    while n > 0:
        d += x[n % r]
        n //= r
    return d[::-1]
```

1. 정렬하기 위해
2. 중복을 제거하기 위해
3. 나머지를 구한 순서가 거꾸로이기 때문에
4. 문자열을 리스트로 바꾸기 위해

<details>
<summary>해설 보기</summary>

- 정답: 3번
- 이유: 나머지는 낮은 자리부터 쌓이므로 최종 결과를 만들 때 순서를 뒤집어야 한다.

</details>

<details>
<summary>문제 35. 다음 코드의 출력 결과는?</summary>

출처: `IMG_8870`, 35번

```python
def reverse(a):
    n = len(a)
    for i in range(n // 2):
        a[i], a[n-i-1] = a[n-i-1], a[i]

x = [1, 2, 3, 4, 5]
reverse(x)
print(x)
```

1. `[5, 4, 3, 2, 1]`
2. `[2, 1, 4, 3, 5]`
3. `[1, 2, 3, 4, 5]`
4. `[5, 4, 3, 1, 2]`

<details>
<summary>해설 보기</summary>

- 정답: 1번
- 이유: 양끝 원소를 교환하는 제자리 역순 알고리즘이므로 전체 순서가 반전된다.

</details>
</details>

<details>
<summary>문제 42. 다음 코드의 출력 결과는?</summary>

출처: `IMG_8872`, 42번

```python
x = '0123456789ABCDEFGHIJKLMNOPQRSTUVWXYZ'

def card_conv(n, r):
    d = ''
    while n > 0:
        d += x[n % r]
        n //= r
    return d[::-1]

print(card_conv(10, 2))
```

1. 10
2. 1010
3. 1001
4. 1111

<details>
<summary>해설 보기</summary>

- 정답: 2번
- 이유: 10을 2진수로 바꾸면 `1010`이다. 이 코드의 핵심은 나머지를 누적한 뒤 마지막에 뒤집는다는 점이다.

</details>
</details>

<summary>문제 36. 다음 역순 알고리즘의 시간복잡도는?</summary>

출처: `IMG_8870`, 36번

```python
def reverse(a):
    n = len(a)
    for i in range(n // 2):
        a[i], a[n-i-1] = a[n-i-1], a[i]
```
