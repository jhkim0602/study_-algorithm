<details>
<summary>문제 33. 다음 코드의 출력 결과는?</summary>

출처: `IMG_8870`, 33번

```python
def max_of(a):
    maximum = a[0]
    for i in range(1, len(a)):
        if a[i] > maximum:
            maximum = a[i]
    return maximum

print(max_of([4, 1, 9, 3, 7]))
```

1. 4
2. 7
3. 9
4. 24

<details>
<summary>해설 보기</summary>

- 정답: 3번
- 이유: 현재 후보 `maximum`을 순차적으로 갱신하면 최종적으로 가장 큰 값 `9`가 남는다.
- 연결 포인트: 최대값 문제는 스캔 패턴과 후보 갱신의 기본형이다.

</details>
</details>

<details>
<summary>문제 34. 배열 길이가 <code>n</code>일 때 비교 연산 <code>a[i] &gt; maximum</code>은 총 몇 번 수행되는가?</summary>

출처: `IMG_8870`, 34번

```python
def max_of(a):
    maximum = a[0]
    for i in range(1, len(a)):
        if a[i] > maximum:
            maximum = a[i]
```

1. `n`
2. `n-1`
3. `n/2`
4. `log n`

<details>
<summary>해설 보기</summary>

- 정답: 2번
- 이유: 반복문이 `1`부터 `len(a) - 1`까지 돌기 때문에 비교는 정확히 `n - 1`번 일어난다.
- 연결 포인트: 스캔 알고리즘은 한 칸씩 훑기 때문에 대체로 선형 시간에 해당한다.

</details>
</details>
