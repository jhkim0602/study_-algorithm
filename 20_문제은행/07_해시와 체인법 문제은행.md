<summary>문제 47. 다음 코드의 출력 결과는?</summary>

출처: `IMG_8873`, 47번

```python
def hash_func(key):
    return key % 5

table = [[] for _ in range(5)]
keys = [7, 12, 17]
for k in keys:
    h = hash_func(k)
    table[h].append(k)

print(table)
```

1. `[[], [], [7, 12, 17], [], []]`
2. `[[], [], [], [7, 12, 17], []]`
3. `[[], [7], [12], [17], []]`
4. `[[], [], [7], [12], [17]]`

<details>
<summary>해설 보기</summary>

- 정답: 1번
- 이유: 세 키 모두 `2`번 버킷으로 들어가므로 체인법에서는 한 리스트 안에 순서대로 저장된다.

</details>

<summary>문제 49. 다음 코드의 출력 결과는?</summary>

출처: `IMG_8873`, 49번

```python
def hash_func(key):
    return key % 5

table = [None] * 5

def insert(key):
    h = hash_func(key)
    while table[h] is not None:
        h = (h + 1) % 5
    table[h] = key

keys = [7, 12, 17]
for k in keys:
    insert(k)

print(table)
```

1. `[None, None, 7, 12, 17]`
2. `[None, None, 17, 7, 12]`
3. `[None, None, 7, 17, 12]`
4. `[7, 12, 17, None, None]`

<details>
<summary>해설 보기</summary>

- 정답: 1번
- 이유: `7`은 2번, `12`는 충돌 후 3번, `17`은 다시 충돌 후 4번에 들어간다. 이는 오픈 주소법의 선형 조사 예시다.

</details>
</details>

학생 상담 신청 정보를 입력받아 저장하고 결과를 출력하는 Spring Boot 웹 프로그램을 작성한다.
프로젝트는 이클립스에서 새로 생성하며, JSP를 사용한다.

C1) 상담 신청 정보는 ConsultInfo 타입 객체에 저장할 것

C2) ConsultInfo 객체에는 학생 이름, 학번, 상담 주제, 연락처를 저장할 것

C3) 상담 정보를 저장하는 List<ConsultInfo> 와 입력 가능한 최대 상담 건수는 DI를 이용하여 ConsultManager 클래스의 속성으로 저장할 것

C4) 최대 상담 건수와 담당 교수명, 안내 메시지는 외부 프로퍼티 파일에 등록할 것

C5) ConsultManager 클래스는 ApplicationContextAware, InitializingBean 인터페이스의 구현 클래스로 작성할 것

C6) 리스트에 추가되는 ConsultInfo 객체는 ConsultManager 클래스에서 스프링 컨테이너로부터 획득할 것

C7) ConsultInfo Bean은 매 요청 처리 시 새로운 객체가 생성되도록 적절한 Scope로 등록할 것

C8) request scope Bean과 session scope Bean을 각각 1개 이상 포함할 것

C9) 동일 인터페이스를 구현하는 알림 Bean 2개 이상을 작성하고, 특정 Bean이 주입되도록 구성할 것

C10) 컨트롤러는 요청을 받아 처리 결과를 JSP로 출력할 것

C11) 상담 신청 입력 화면과 결과 출력 화면을 구성할 것

C12) 초기화 메서드와 종료 메서드가 동작하도록 구성할 것

C13) ConsultManager Bean은 destroyMethod에 의해 지정된 메서드에서 전체 상담 결과를 출력할 것

C14) 보조 Bean 1개 이상은 XML 설정 방식으로 등록하여 사용할 것

C15) 상담 신청 처리 메서드에 대한 공통 기능은 AOP로 적용할 것

C16) AOP 설정은 XML 기반으로 작성하고, 메서드 실행 전후 동작 또는 실행 시간 측정 결과를 확인할 수 있어야 한다
