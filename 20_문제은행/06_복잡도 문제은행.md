<details>
<summary>문제 34. 배열 길이가 <code>n</code>일 때 비교 연산은 총 몇 번 수행되는가?</summary>

출처: `IMG_8870`, 34번

```python
def max_of(a):
    maximum = a[0]
    for i in range(1, len(a)):
        if a[i] > maximum:
            maximum = a[i]
```

1. `n`
2. `n-1`
3. `n/2`
4. `log n`

<details>
<summary>해설 보기</summary>

- 정답: 2번
- 이유: 선형 스캔 알고리즘은 한 칸씩 보기 때문에 `n - 1`번 비교한다.

</details>
</details>

<details>
<summary>문제 36. 다음 역순 알고리즘의 시간복잡도는?</summary>

출처: `IMG_8870`, 36번

```python
def reverse(a):
    n = len(a)
    for i in range(n // 2):
        a[i], a[n-i-1] = a[n-i-1], a[i]
```

1. `O(1)`
2. `O(log n)`
3. `O(n)`
4. `O(n^2)`

<details>
<summary>해설 보기</summary>

- 정답: 3번
- 이유: 절반 반복이어도 선형 계열이다.

</details>
</details>

<details>
<summary>문제 39. 다음 선형검색에서 <code>key</code>가 존재하지 않을 때 비교 횟수는?</summary>

출처: `IMG_8871`, 39번

```python
def seq_search(a, key):
    for i in range(len(a)):
        if a[i] == key:
            return i
    return -1

seq_search([2, 4, 6, 8, 10], 7)
```

1. 4
2. 5
3. 6
4. `log 5`

<details>
<summary>해설 보기</summary>

- 정답: 2번
- 이유: 실패 시 전체 길이만큼 비교한다.

</details>
</details>

<details>
<summary>문제 45. 다음 코드의 시간복잡도로 가장 적절한 것은?</summary>

출처: `IMG_8872`, 45번

```python
for n in range(2, N):
    for i in range(2, n):
        if n % i == 0:
            break
```

1. `O(log n)`
2. `O(n)`
3. `O(n log n)`
4. `O(n^2)`

<details>
<summary>해설 보기</summary>

- 정답: 4번
- 이유: 가장 단순한 상한으로 보면 이중 반복 구조다.

</details>
</details>
