# 자료구조 학습 Vault

이 Vault는 **강의 출처 중심이 아니라 개념과 실습 중심**으로 다시 짠 학습 폴더다.  
노트 하나 안에 배경 설명, 의사코드, 실제 Python 코드, 실수 포인트를 같이 넣어 두었기 때문에 파일을 왔다 갔다 하지 않아도 된다.

- 시작 노트: [[허브 - Home]]

루트 구조는 최대한 단순하게 줄였다.

- `00_허브`: 시작 노트 하나
- `10_개념_노트`: 실질적인 공부 본문
- `20_복습_자료`: 용어 정리, 질문 은행, 체크리스트
- `assets`: 원본 영상 보관
